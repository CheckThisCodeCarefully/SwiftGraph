  • djikstra(_:_:)

    Finds the shortest paths from some route vertex to every other vertex in the graph. Note this doesn’t yet use a priority queue, so it is very slow.

    params
    graph The WeightedGraph to look within.
    params
    root The index of the root node to build the shortest paths from.

    Declaration

    Swift

    func djikstra<T: Equatable, W: protocol<Comparable, Summable>> (graph: WeightedGraph<T, W>, root: Int) -> ([W?], [Int: WeightedEdge<W>])

    Return Value

    Returns a tuple of two things: the first, an array containing the distances, the second, a dictionary containing the edge to reach each vertex. Use the function pathDictToPath() to convert the dictionary into something useful for a specific point.
